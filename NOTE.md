## 开发指南
1. 项目构建
* 脚手架
`npm install -g create-react-app`
2. 安装项目
`create-react-app bike`
3. 运行项目
```
>> cd bike
>> yarn start
```
4. 安装插件
`yarn add react-router-dom axios less-loader`
5. 暴露配置
`yarn eject`
